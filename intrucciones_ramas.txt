CAMVIAR A SU RAMA: 
git checkout -b [nombre de su rama]

git add .
git commit -m "MENSAJE DE LO QUE SE ESTA ACTUALIZANDO EN EL COMMIT"

git push --set-upstream origin [nombre de su rama]

git push origin
